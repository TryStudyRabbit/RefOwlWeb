Release Notes

We're constantly working to improve RefOwl with new features, bug fixes, and performance enhancements. Check this page to stay updated on the latest changes.

Version 1.0.1
April 21, 2025

Improvements

  • Reduced extension permissions to only require what's needed
  • Updated owl logo across all extension assets

Bug Fixes

  • Removed unused "scripting" permission
Version 1.0.0
April 15, 2025

Initial Release

  • Auto-fill metadata extraction from web pages
  • APA, MLA, and Chicago citation formatting
  • Save and manage citations in a library
  • Copy citations to clipboard
  • Basic citation editing capabilities
  • Local storage of all citation data for privacy
Version 1.3.0
April 5, 2025

New Features

  • Added support for Chicago style (17th Edition)
  • Implemented tagging system for better citation organization
  • Added BibTeX export format option
  • New filter options in the citation library

Improvements

  • Enhanced metadata extraction accuracy for journal articles
  • Improved date detection algorithm
  • Updated APA style to latest guidelines
  • More responsive UI for smaller screens

Bug Fixes

  • Fixed issue with incorrect page number formatting in MLA style
  • Resolved problem with author name detection for multiple authors
  • Fixed export functionality on some Chromium-based browsers
  • Corrected handling of special characters in titles
Version 1.2.0

New Features

  • Added MLA (9th Edition) support
  • Implemented search functionality in citation library
  • Added ability to batch edit multiple citations
  • New Word document export option

Improvements

  • Faster metadata extraction from webpages
  • More accurate DOI detection
  • Updated extension icons and UI elements
  • Better handling of complex author names

Bug Fixes

  • Fixed issue with citation sorting in the library
  • Resolved date formatting inconsistencies
  • Fixed popup positioning on some websites
  • Corrected handling of URLs with special parameters
